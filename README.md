# Dr Jeremy Sun Website

Vercel-ready Next.js personal website starter.

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel
Connect this folder to a GitHub repository, then import the repo in Vercel.
